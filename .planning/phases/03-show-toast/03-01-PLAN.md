# Plan: show_toast 端到端验证

**Objective:** 验证 C++ Agent 调用 Java Tool 的完整流程

## Requirements
- TOOL-04

## Tasks

### T1: 安装 APK 到设备
- 连接 Android 设备
- 执行 `adb install -r app/build/outputs/apk/debug/app-debug.apk`

### T2: 触发 Agent 调用 show_toast
- 打开应用
- 发送消息 "显示一个 Toast 测试消息"
- 或发送消息触发 Agent 调用 tool

### T3: 验证 Toast 显示
- 检查设备屏幕是否有 Toast 显示
- 检查 logcat 输出

## Verification Criteria

1. APK 安装成功
2. 发送消息后 Agent 响应
