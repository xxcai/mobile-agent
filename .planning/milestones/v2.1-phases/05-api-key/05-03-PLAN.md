在 AndroidToolManager 中添加动态生成 tools.json 的逻辑：

1. 添加 `generateToolsJson()` 方法：
   - 遍历 tools Map (name -> ToolExecutor)
   - 对每个 Tool 调用:
     - getName() → enum 列表
     - getDescription() + getArgsDescription() → description 字段
     - getArgsSchema() → parameters Schema
   - 拼接为 call_android_tool 格式的 JSON

2. 在 initialize() 方法末尾调用 generateToolsJson() 生成 JSON

3. 通过 NativeMobileAgentApi.setToolsJson() 传递给 C++ 层

生成的 JSON 格式：
```json
{
  "version": 2,
  "tools": [
    {
      "type": "function",
      "function": {
        "name": "call_android_tool",
        "description": "调用 Android 设备功能。可用功能:\n- show_toast: 显示Toast消息, 参数: message, duration\n...",
        "parameters": {
          "type": "object",
          "properties": {
            "function": {
              "type": "string",
              "description": "要调用的功能名称",
              "enum": ["show_toast", "display_notification", ...]
            },
            "args": {
              "type": "object",
              "description": "功能参数，JSON 对象格式"
            }
          },
          "required": ["function", "args"]
        }
      }
    }
  ]
}
```
