# Gap 1: 点击外部区域无法收起
## Root Cause
contentArea设置MATCH_PARENT覆盖了整个mRootLayout可视区域，事件被子View消费，根布局OnClickListener无法收到

## Fix
在根布局上添加透明遮罩View捕获外部点击

# Gap 2: 悬浮球不恢复显示
## Root Cause
sendShowFloatingBallBroadcast()在onDestroy()中调用，Activity销毁时广播可能无法被接收

## Fix
在finish()方法中发送广播，确保Activity销毁前发出
</context>

    - 点击mRootLayout的空白区域（遮罩层），Activity收起

1. 在创建contentArea之后，创建一个透明遮罩View
2. 遮罩View设置：
   - 背景色为透明 (0x00000000)
   - MATCH_PARENT宽高
   - 点击事件监听器，点击时调用finish()
3. 将遮罩View添加到mRootLayout中（在contentArea之后添加，确保覆盖contentArea以外的区域）
4. 确保contentArea设置点击事件时不拦截向上传递的事件，或者直接去掉contentArea的点击监听

关键修改：
- 添加 setupMaskView() 方法创建透明遮罩
- 遮罩View的OnClickListener调用finish()
- 遮罩View必须在contentArea之后add到mRootLayout，这样它会覆盖contentArea以外的区域
