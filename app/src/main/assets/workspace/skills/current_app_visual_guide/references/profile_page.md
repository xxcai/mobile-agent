# Profile Page

基于当前前台界面整理的“个人中心”页结构。

## 页面主体结构

“个人中心”页是当前应用中的个人信息与个人服务聚合页，整体通常分为上下两部分：

- 上半部分是个人信息区域
- 下半部分是按主题分组的功能入口区域

## 个人信息区域

页面顶部通常展示：

- 页面标题“个人中心”
- 用户姓名
- 工号或员工编号
- 头像或个人标识区域

这一部分主要用于表达“当前是谁的个人中心”，也构成页面身份感最强的区域。

## 分组功能区

个人中心下半部分通常以分组堆叠的方式组织个人服务入口。

当前界面中可见的分组包括：

- 常用
- HR服务
- 行政服务
- 其他

每个分组下方通常是宫格形式的功能入口，并可能带有“更多”入口。

## 当前可见入口示例

### 常用

- 云空间
- 云笔记
- 充值
- 卡包
- 云名片
- 收藏
- 求助
- 设置

### HR服务

- 我的信息
- 我的档案
- 考勤
- 薪酬

### 行政服务

- 差旅
- 车辆
- 物业
- 人事

### 其他

- 酒店标准
- 用户安全中心
- 发票抬头
- 我的知识
- 我的项目

## 页面功能概括

“个人中心”页可以理解为当前应用中的个人服务首页。
它既承载个人信息展示，也承载与个人相关的常用能力、HR能力、行政能力和其他辅助服务入口。
