# Home Quick Actions Popup

基于当前前台界面整理的“主界面快捷操作弹层”结构。

## 结构位置

这是一个覆盖在主界面之上的菜单型浮层，通常出现在右上区域，以纵向列表形式展开。

它本身不替代主界面，而是在主界面上临时承载一组快捷动作入口。

## 当前可见快捷操作

当前界面中可见的条目包括：

- 创建群聊
- 语音会议
- 视频会议
- AI听记
- 投影
- 扫一扫
- 创建个人助理

## 结构特征

- 每一行对应一个独立快捷操作
- 条目以垂直堆叠方式排列
- 整体更像“操作菜单”而不是信息展示卡片

## 功能概括

这个弹层可以理解为主界面上的快捷操作集合。
它集中放置了沟通、会议、设备协同、扫描和智能助理相关的高频入口，方便用户从主界面快速发起动作。
